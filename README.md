# music-sympathy

Music Clustering for ICCS413 Term Project

## Development

Create virtualenv (using python 3)
```bash
$ virtualenv -p python3 venv
```

Activate virtualenv
```bash
$ source venv/bin/activate
```

Install packages
```bash
$ pip install -r requirements.txt
```

Deactivate virtualenv
```bash
$ deactivate
```
